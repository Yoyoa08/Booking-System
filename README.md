# Booking Management System

A Python-based technical business solution built with the **Streamlit** framework.

## Features
* User-friendly dashboard for managing bookings.
* Real-time data visualization.
* Streamlined interface for client management.

## Tech Stack
* **Python**
* **Streamlit**

## How to Run
1. Install requirements: `pip install streamlit`
2. Run the app: `streamlit run main.py`
